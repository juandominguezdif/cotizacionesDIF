DIF Cotizaciones Web v9

Cambios v9:
- La caja de comentario ya no se llena automáticamente al abrir la web.
- La web ya no consulta el comentario guardado en SharePoint al iniciar ni al cambiar FCL/LCL/Aéreo o idioma.
- El comentario se usa como entrada nueva para enviar por Power Automate.
- Después de enviarlo, la caja se limpia.
- El flujo Power Automate puede seguir guardando/enviando el comentario por correo según la configuración actual.

Abrir index.html en el navegador o subir este folder a GitHub Pages.
